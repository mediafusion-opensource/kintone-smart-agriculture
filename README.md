##  概要

**スマート農業労務会計システム**は、kintoneの柔軟なデータ管理機能とMicrosoft 365のスマートフォン対応機能を組み合わせた、農業事業者向けの業務管理システムです。

Media Fusion Inc.の社会貢献活動として、中小規模の農業事業者の皆様に**無償で提供**しています。

---

##  特徴

-  **計画・勤怠管理** - 作付け計画、カレンダー管理、スマホでの作業記録
-  **収穫・在庫・販売管理** - 収穫量記録、在庫トラッキング、販売履歴
-  **BI分析・経営支援** - Power BIダッシュボード、収益性分析
-  **スマートフォン対応** - 現場からリアルタイムでデータ入力
-  **コスト削減** - ライセンス費用を最小限に

---

##  システム構成

```
┌─────────────┐     ┌─────────────────┐     ┌─────────────┐
│   kintone   │ ←→  │   SharePoint    │ ←→  │   スマホ     │
│ データ管理   │     │   データ同期     │     │  MS365 App  │
└─────────────┘     └─────────────────┘     └─────────────┘
```

| コンポーネント           | 提供形態        | 対象         |
|-------------------------|-----------------|-------------|
| kintoneデータ管理アプリ | 無償（本リポジトリ） | すべてのユーザー  |
| MS365スマホアプリ      | 有償（別途契約）  | 10人超のユーザー |
